# Расписание СГАУ

Веб-приложение для просмотра расписания занятий Самарского государственного аэрокосмического университета (СГАУ). Данные берутся напрямую с сайта [ssau.ru](https://ssau.ru/rasp).

## Возможности

- Поиск расписания по номеру группы или ФИО преподавателя
- Просмотр расписания на любую учебную неделю
- Навигация между неделями
- Скачивание расписания в формате PDF
- Тёмная тема (сохраняется между сессиями)
- Адаптивный дизайн для мобильных устройств

## Технологии

**Backend:**
- Node.js + Express
- `node-html-parser` — парсинг HTML с ssau.ru
- `xhr2` — HTTP-запросы на сервере

**Frontend:**
- jQuery 3.7.1
- html2pdf.js — экспорт в PDF
- CSS с адаптацией под мобильные устройства

## Установка и запуск

```bash
# Установить зависимости
npm install

# Запустить сервер
node server.js
```

Открыть в браузере: [http://localhost:3000](http://localhost:3000)

## Структура проекта

```
├── server.js                 # Серверная часть (Express)
├── ListGroupsTeachers.json   # Справочник групп и преподавателей
├── package.json
└── public/
    ├── index.html            # Главная страница
    ├── style.css             # Стили
    └── app.js                # Клиентская логика (jQuery)
```

## Использование

1. Введите номер группы (например `6411-010302D`) или фамилию преподавателя в строку поиска
2. Выберите нужную запись из выпадающего списка
3. Расписание загрузится автоматически
4. Используйте кнопки навигации для переключения недель
5. Нажмите **Скачать PDF** для сохранения расписания
